Distributed Kalman Filtering With Reduced Transmission Rate

In this paper, two distributed Kalman filtering algorithms are proposed. The first algorithm merely requires communication of each local sensor node with the fusion center in every other time step. The second algorithm even allows for a lower communicate rate. Both algorithms apply event-based communication to compute consistent estimates and to reduce the estimation error for a fixed communication rate.

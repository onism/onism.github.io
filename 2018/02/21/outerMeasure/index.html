I am working with Jeremie Houssineau at NUS # Bayesian data assimilation based on a family of outer measures (2016) In this article, they study a representation of uncertainty that is completely within the standard framework of probabilistic measure theory. ## Uncertainty and Probability measures

Example: Consider a discrete-time joint process \(X_n,Y_n\) assumed to be a hidden Markov model and consider the case where realisations of \(Y_n\) of the observation process are not received directly but are instead known to be in some subset \(A_n\) of the observation space.

There is a need for the introduction of additional concepts that could account for these non-informatives types of knowledge, and which would in turn enable data assimilation to be performed in more general spaces.

Fundamental Concepts

Outer measure

An outer measure on \(\textbf{E}\) is a function verifying the following conditions:

  1. Null empty set
  2. Monotonicity
  3. Countable sub-additivity

Representation of uncertainty

Defining outer measures on measureable functions is then preferred since it is more general. It is better to understand random variables as solely describing randomness with another concept needed to describe uncertainty in general. The concept of uncertain variable is introduced in order to describe a variable about which \(P\) gives information.

HMM and outer measures

Jeremie propose the following approach: the uncertainty about a collection \(\{X_t \}\) of uncertain variables has the Markov property if for any \(t\) it holds that

\[\bar{P}(\varphi)(x_{0:t-1}) = \bar{P}(x_{t-1})\]

In some case, the uncertain variables will represent an unknown but non-random change in the model, e.g. a plane manoeuvring or a pedestrian changing direction.

Detection and estimation of partially-observed dynamical systems: an outer-measure approach

We consider a set \(\Omega_u\) containing the possible states for the deterministic phenomena of interest. A mapping \(X\) from the set \(\Omega_u\) of outcomes to a set \(E\) is referred to as an E-valued deterministic uncertain variable.

Let \(\bar{P}\) be an o.p.m describing an uncertain variable \(X\) on \(E\) abd ket \(\bar{P}^{'}(\cdot|X = x)\) be a conditional o.p.m. describing another uncertain variable \(X^{'}\) on \(F\) conditioned on \(X = x\) for any \(x \in E\), then the marginal o.p.m. is characterised by

\[\bar{P}^{'} (\varphi) = \int ||f^{'}(x^{'}|\cdot) \cdot f||_{\infty} P(df) P^{'}(df^{'}|X)\]

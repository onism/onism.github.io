A Stable Particle Filter in High-Dimensions

In high dimensionas, standard particle filters can have a cost that is exponential in ddd for the algorithm to be stable in an appropriate sense. This paper proposed a new particle filter, called space-time particle filter, for a specific family of state space models in discrete time.

The authors break the kkk-th time step of the particle filter into τk,d\tau_{k,d}τ​k,d​​ space-steps and run a system of NNN independent particle filters for these steps.

The Space-Time Particle Filter

The authors develop an algorithm that combines a local filter running ddd space-step using MdM_dM​d​​ particles, with a global filter making time-steps and uses NNN particles.

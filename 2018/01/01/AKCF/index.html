Distributed Agent-based Dynamic State Estimation over a Lossy Network

Dynamic state estimation in a large CPS presents some unique challenges. A typical example is the smart electric power distribution system with thousands of end-uses. For such a system, the dimension of corresponding state vector is quite large presenting high computational as well as communication burden for the sensors. In conventional distributed Kalman filter setup, each sensor has to regularly store and update the global state vector and estimation error covariance matrix. Thus, it may be impractical to track the high dimensional state vector in its entirety at each communicable sensor.

Some state elements may even be coupled to its local observation space. Under these circumstances, each sensor may be relieved to track only the pertinent state elements, thus reducing thes ize of error covariance matrices and subsequent reduction in computational requirements.

In this paper, an agent-based formulation of KCF and DKF is proposed. Each agent has access to a distinct set of measurements, that are coupled to a subset of global state elements.

Unlike conventional distributed Kalman filtering, a particular agent \(k\) attempts to estimate only the local state vector \(x_{k,k}\) instead of \(x_t\).
